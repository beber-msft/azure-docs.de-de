```Bash
pip install azure-cognitiveservices-speech
```

> [!TIP]
> Unter macOS müssen Sie möglicherweise den folgenden Befehl ausführen, damit der obige pip-Befehl funktioniert:
> ```Bash
> python3 -m pip install --upgrade pip
> ```
